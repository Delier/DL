在现有项目种初始化仓库：git init
初次克隆仓库，工作目录的所有文件都属于已跟踪文件
add将文件放入暂存区
查看已暂存和未暂存的修改：git diff
查看已暂存将要添加到下次提交的内容：git diff --staged
移除文件是移除暂存区的文件：git rm [要移除的文件名]
克隆仓库：git clone [url]
克隆远程仓库并且定义自己本地仓库的名字：git clone [url] [自定义的仓库名称]
取消暂存的文件:git reset HEAD [取消暂存的文件名]
撤销对文件的修改:git checkout -- [文件] 会还原成上次提交时的样子
创建分支：git checkout -b [分支名称]
查看提交历史：git log


查看所有远程仓库：git remote
添加远程仓库：git remote add [自定义远程仓库的简写] [url]
拉取分支：git pull origin [分支名]
推送到远程仓库：git push origin [分支名]
查看某个远程仓库：git remote show [仓库]
远程仓库的重命名:git remote rename [旧名称] [新名称]
远程仓库的移除:git remote rm [仓库]
